# 🌾 Agroassistente

**Agroassistente** é uma aplicação web interativa desenvolvida em **Python** com **Streamlit**, focada em facilitar o trabalho diário de **agricultores**, **engenheiros agrônomos** e **técnicos agrícolas**. O objetivo do projeto é oferecer ferramentas práticas, acessíveis e automatizadas para atividades recorrentes no campo, como adubação, calagem, semeadura, uso de defensivos, entre outras.

---

## 🚀 Funcionalidades atuais

✅ Implementadas:

- **Cálculo de Calagem**
- **Cálculo de Saturação por Bases (V%)**
- **Recomendação de Adubação com base em análise de solo**

---
